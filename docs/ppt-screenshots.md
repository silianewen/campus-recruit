## 截图规范

- **分辨率**：1920×1080 或 1280×720，不要糊
- **格式**：PNG（清晰）不要 JPG（有损）
- **窗口**：浏览器全屏（F11），隐藏书签栏（Ctrl+Shift+B），隐藏扩展图标
- **数据隐私**：
  - 手机号打码：138****8000
  - HR 密码打码
  - 真实姓名如果涉及可以保留（测试数据）
- **统一缩放**：截图后所有图缩放到相同宽度（如 1280px），PPT 排版整齐

## 汇报 PPT 结构建议（10 页）

```
P1  封面（标题 / 汇报人 / 日期）
P2  项目背景（3-5 行，痛点）
P3  解决方案概览（用 #01 截图）
P4  学生端流程（#02 + #03 + #04 三张拼图）
P5  测评模块（#05 MBTI 结果）
P6  HR 后台（#06 + #07）
P7  数据看板（#08 + #09 上下排）
P8  移动端体验（#10）
P9  成本与时间（一句话："月成本 < 10 元，1 周交付"）
P10 后续规划 + 二维码（演示视频 QR + 平台 QR）
```

每页文字 ≤ 30 字，图为主、文字为辅。
