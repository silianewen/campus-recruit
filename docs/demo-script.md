# 演示视频脚本（2 分钟）

> 用途：汇报当天播放 / 提交给领导看
> 录制工具：Windows 自带 `Win + G`（Xbox Game Bar）或 OBS
> 分辨率：1920×1080 或 1280×720
> 时长：控制在 1:50 ~ 2:10

## 录制前准备

1. **清空现场**：关掉其他标签页、关通知、关屏保
2. **准备好测试数据**：
   - 一个手机号 `13800138000` 已经投过前端简历（用 9 步验证的）
   - 跑过 `node scripts/seed-test-data.mjs` 有 50 条数据
3. **HR 密码** 背下来（输入用）
4. **浏览器**：用 Chrome 或 Edge，缩放 100%，开 F11 全屏
5. **窗口**：1920×1080 屏幕最佳，笔记本外接显示器更好

## 录制技巧

- 🎯 **鼠标移动要慢**，关键按钮 hover 1-2 秒再点
- 🔇 **Tab 切换前停 1 秒**，给观众看当前画面
- ✂️ **剪辑掉所有加载白屏**：录完用剪映/PR 加速
- 🎤 **旁白后期录**：先静默操作 + 录屏，后期配音（避免环境噪音）
- ⏱️ **总时长严控 2 分钟**：超过就剪

## 后期剪辑清单

- [ ] 加片头（3 秒，标题"2026 校招投递平台演示"）
- [ ] 关键步骤加文字标注（如 "扫码 → 30 秒投递"）
- [ ] HR 后台密码打码
- [ ] 手机号打码（中间 4 位 `****`）
- [ ] 加片尾（5 秒，二维码 + 联系方式）

## 输出格式

- MP4 / H.264
- 1080p
- 文件大小 < 50MB（汇报现场投屏不会卡）
- 命名：`校招平台演示-2026-07-XX.mp4`
